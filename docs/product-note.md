# Product Note

## Problem

Beginners practicing sign language handshapes often do not get immediate feedback about hand position, finger shape, or palm direction.

## Prototype idea

A camera-based coach that detects hand landmarks and gives simple feedback while the user practices.

## Scope

V0 focuses only on basic ASL handshapes A, B, and L.

## Non-goals

- Full sign language translation
- Real-time conversation translation
- Grammar correction
- Replacing Deaf teachers or certified learning resources
- Supporting multiple sign languages in V0

## Future ideas

- More ASL alphabet letters
- Movement-based signs
- Saved practice sessions
- Side-by-side reference videos
- Better scoring model
- Multi-language sign packs such as ASL, BSL, Ukrainian Sign Language, etc.
- Community-reviewed sign references
- Accessibility-focused UX testing
